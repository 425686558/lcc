lcc, a Retargetable Compiler for ANSI C

lcc is a retargetable compiler for C as defined by the ANSI Standard X3.159-1989. It generates code for the SPARC, MIPS R3000, and Intel 386 and its successors. lcc is in production use at Princeton University and AT&T Bell Laboratories. The man page gives usage details.

lcc is available for anonymous ftp from ftp.cs.princeton.edu in pub/lcc; the README file gives the details. Here's what else is available:

  • lcc version 3.3 is now available; it fixes all of the errors listed in the errata for the first printing of the book; For details see the LOG file. Version 3.4 is imminent: it adds support for dynamic linking on the MIPS and fixes a nasty bug in 3.3's generated code for switch statements.
  • A list of frequently asked questions.
  • The installation guide describes the steps necessary to install lcc.
  • How to report bugs; see the frequently asked questions before reporting bugs.
  • The source code for the current version.
  • User-contributed software is available in pub/lcc/contrib; its README lists the contributions and gives instructions for contributors.
  • Additional documentation about lcc and a bibliography of lcc-related papers.
lcc is described in the book, A Retargetable C Compiler: Design and Implementation (Benjamin/Cummings, 1995, ISBN 0-8053-1670-1). This book is a detailed tour of lcc's source code. The ftp distribution in pub/lcc and the companion diskette both hold the source code for the complete compiler, including the code generators for the SPARC, MIPS R3000 and Intel 386, and the code-generator generator that produced them. See the preface and table of contents for more details, and for an unbiased opinion, see the 4/28/95 review posted to comp.compilers. An ASCII version of the book's index is also available on ftp.cs.princeton.edu in pub/lcc/contrib/bookidx.txt; download this file to search the index online.

There is an lcc mailing list for general information about lcc. To be added to the list, send a message with the 1-line body 

subscribe lcc
to majordomo@cs.princeton.edu. This line must appear in the message body; `Subject:' lines are ignored. To learn more about mailing lists served by majordomo, send a message with the body `help' to majordomo@cs.princeton.edu.

Mail sent to lcc@cs.princeton.edu to is forwarded to everyone on the mailing list. There is also an lcc-bugs mailing list for reporting bugs; mail addressed to lcc-bugs@cs.princeton.edu is forwarded to everyone on this list.
